Hayai

Hayai is a free and open-source manga reader for Android 6.0 and above. This version builds upon TachiyomiSY, enhancing performance, usability, and features while maintaining compatibility with updates from the main app.



Features

Core Features (Inherited from Mihon/TachiyomiSY)

Online reading from a variety of sources

Local reading of downloaded content

A configurable reader with multiple viewers, reading directions, and other settings

Tracker support: MyAnimeList, AniList, Kitsu, MangaUpdates, Shikimori, and Bangumi support

Categories to organize your library

Light and dark themes

Scheduled library updates for new chapters

Local and cloud backup support


Hayai-Specific Enhancements

Way faster library refreshes

Fixed image optimization for better quality

Way faster downloads with configurable threads and concurrency

Complete UI redesign for improved aesthetics and usability

Fixes to auto-update ensuring smoother updates

Fuzzy search for better discovery

New themes to customize your experience

Faster migrations when switching sources

Quality of life features to improve the overall experience
